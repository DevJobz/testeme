# QuestionAI - Gerador de Questões Inteligente

## 📋 Descrição

O QuestionAI é um aplicativo web completo para geração de questões inéditas a partir de arquivos PDF ou textos inseridos diretamente. O sistema utiliza inteligência artificial para criar questões personalizadas com diferentes níveis de dificuldade e tipos.

## ✨ Funcionalidades Principais

### 1. Sistema de Autenticação
- **Cadastro completo** de usuários com validação
- **Login seguro** com persistência de sessão
- **Perfil de usuário** personalizável com foto, informações pessoais e preferências

### 2. Geração de Questões
- **Upload de PDF** para extração automática de conteúdo
- **Inserção de texto** manual com contador de caracteres/palavras
- **Configurações personalizáveis**:
  - Quantidade de questões (5, 10, 15, 20)
  - Nível de dificuldade (Fácil, Médio, Difícil, Misto)
  - Tipo de questão (Múltipla Escolha, Verdadeiro/Falso, Dissertativa, Misto)
  - Área de foco (Geral, Conceitos Principais, Detalhes Específicos, Aplicação Prática)

### 3. Gerenciamento de Questões
- **Visualização** de todas as questões geradas
- **Sistema de filtros** por status, dificuldade e tipo
- **Busca avançada** por conteúdo
- **Resposta interativa** com feedback imediato
- **Sistema de favoritos** para questões importantes

### 4. Relatórios e Análises
- **Dashboard** com estatísticas gerais
- **Gráficos de desempenho** por dificuldade
- **Progresso ao longo do tempo**
- **Análise de acertos e erros**
- **Métricas detalhadas** de estudo

### 5. Armazenamento Local
- **LocalStorage** para persistência de dados
- **Backup automático** de informações
- **Exportação/Importação** de dados
- **Sincronização** entre sessões

### 6. Funcionalidades Extras
- **Modo escuro/claro** para melhor experiência
- **Sistema de notificações** para lembretes
- **Atalhos de teclado** para navegação rápida
- **Busca global** (Ctrl/Cmd + K)
- **Interface responsiva** para desktop e mobile

## 🚀 Como Usar

### Primeiro Acesso
1. Acesse o aplicativo no navegador
2. Clique em "Cadastre-se" para criar uma conta
3. Preencha seus dados pessoais
4. Faça login com suas credenciais

### Gerando Questões
1. Vá para "Gerar Questões"
2. Escolha entre:
   - **Upload PDF**: Arraste e solte ou selecione um arquivo
   - **Inserir Texto**: Cole ou digite o conteúdo manualmente
3. Configure as opções de geração
4. Clique em "Gerar Questões"

### Respondendo Questões
1. Acesse "Minhas Questões"
2. Use os filtros para encontrar questões específicas
3. Clique em uma questão para respondê-la
4. Receba feedback imediato sobre sua resposta

### Acompanhando Progresso
1. Visite o "Dashboard" para visão geral
2. Acesse "Relatórios" para análises detalhadas
3. Acompanhe suas estatísticas e evolução

## ⌨️ Atalhos de Teclado

- **Ctrl/Cmd + K**: Busca global
- **Ctrl/Cmd + N**: Gerar novas questões
- **Ctrl/Cmd + D**: Ir para dashboard
- **Ctrl/Cmd + /**: Mostrar atalhos
- **Escape**: Fechar modais

## 🆘 Suporte

### Problemas Comuns
1. **Questões não aparecem**: Verifique se há conteúdo suficiente no texto/PDF
2. **Login não funciona**: Limpe o cache do navegador
3. **PDF não carrega**: Verifique se o arquivo não está corrompido

### Backup de Dados
- Use "Exportar Dados" no perfil para fazer backup
- Mantenha seus arquivos de backup seguros
- Use "Importar Dados" para restaurar informações
